COMMENT TRICHER !!!

Lancez le jeu en passant par la commande "C:\Program Files\MaxPayne\MaxPayne.exe" -developer.
Pendant la partie, appuyez sur F12 pour afficher la console puis entrez l'un des codes suivants.

CODER Debug mode
GETALLWEAPONS Toutes les Armes
GETBASEBALLHAT Avoir une casquette de base-ball
GETINFINITEAMMO Munitions infinies
GETINGRAMDUAL Avoir deux Ingrams avec le plein de munitions
GETMP5 Avoir le MP5
GETM79 Avoir le M-79
GETPAINKILLERS Avoir les Pain Killers
GOD God mode
JUMP # Régler la hauteur de saut
MORTAL Désactiver le Gode mode
NOCLIP Pas de clipping
NOCLIP_OFF Désactive le mode sans clipping


Sauter la séquence de début

Lancez le jeu en passant par la commande "C:\Program Files\MaxPayne\MaxPayne.exe" -skipstartup.

Vous pouvez combiner cette manipulation avec toutes les autres.

Vous pouvez combiner cette manipulation avec toutes les autres, mais attention !

COMMENT COMPILER SA MAP

Si vous avez essayé MaxEd, vous avez sûrement remarqué que c'est très très très dure,
surtout pour avoir un fichier final au format *.mpm
(chez moi, ça bug toujours avant la fin !),
alors voici un bidouille qui semble plus complète pour créer un Mod, et plus efficace !
Le seul bémol, c'est que ça se passe sous dos, voilà comment faire :

créez un dossier "MaxUnpack" n'importe ou, et mettez-y :
x_data.ras et x_french.ras trouvés dans le répertoire de MP
plus les deux fichiers de rasmaker (un outil Max-fx).
passez sous dos, allez dans votre dossier "MaxUnpack" et tapez le code suivant :
rasmaker -x x_data.ras
puis :
rasmaker -x x_french.ras

Rasmaker aura alors "dépacké" la plupart des fichiers de MP,
et vous pouvez supprimer les fichiers x_machin.ras de votre répertoire "MaxUnpack".
Regardez les, vous avez à peu près tout
(fichiers sons, menus, console, models, textures, paramètre a gogos,...).
Maintenant, faut un peu mettre les mains dans le cambouis pour transformer le tout,
mais vous pouvez commencer par transformer par exemple des skins.
Une fois que vous avez édité des fichiers, vous pouvez supprimez tout les fichiers
qui sont dans des répertoires ou vous n'avez rien touché
(toujours dans le répertoire "MaxUnpack" bien sur).

Maintenant, revenez au dos, dans le répertoire "MaxUnpack", et tapez :

rasmaker -a data [répertoire de MP][nom de votre mod + ".mpm].
Par exemple ça donnera :

rasmaker -a data D:jeuxmaxpay~1ultimamod.mpm
"maxpay~1" voulant dire "MaxPayne" mais c trop long pour ms-dos
Maintenant, lancez MP, et dans la premiere boite de dialogue, choisissez "ultimamod.mpm"
(par exemple toujours) et faites "Jouer".
Si vous avez tout bien fait, et ke vous n'avez pas créé d'erreur en modifiant des fichiers
(pour moi, c'est ce qui c'est passé !), ce sera bon !
